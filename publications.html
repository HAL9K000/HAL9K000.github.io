Tools

  • slick.py

    Slick is a lean Python wrapper library for Cupy libraries, designed to provide zero-change GPU acceleration for Numpy/Scipy via Cupy with automatic, intelligent fallbacks. Designed for seamless integration with existing Numpy/Scipy code, allow for drop-in replacement of Numpy/Scipy calls with minimal refactoring. A more in-depth description is provided here.
